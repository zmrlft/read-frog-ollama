陪读蛙专注于深度语言学习，不仅仅是一个翻译工具。它专为不同语言水平的用户设计，是一个全面的阅读工具。

目前，它支持对文章中的单词、短语和句子进行详细解释。未来将添加词汇复习系统、用户语言水平分析和电子书支持等功能。

## 与传统翻译工具的对比

| 功能     | 陪读蛙                         | 沉浸式翻译          | 浏览器内置翻译  |
| -------- | ------------------------------ | ------------------- | --------------- |
| 使用场景 | 深度阅读解释                   | 简单翻译            | 简单翻译        |
| 翻译质量 | ✅ AI 翻译                     | ✅ AI 翻译          | ❌ 质量较差     |
| 翻译引擎 | 🟡 OpenAI, DeepSeek (未来更多) | ✅ 20+ 引擎         | 🟡 单一引擎     |
| 深度学习 | ✅ 适合语言学习者              | ❌ 无               | ❌ 无           |
| 价格     | ✅ 免费                        | 🟡 部分功能付费     | ✅ 免费         |
| 开源     | ✅ 开源，可定制                | ❌ 闭源             | ❌ 闭源         |
| 隐私保护 | ✅ 本地数据存储                | 🟡 存在数据泄露风险 | ✅ 本地数据存储 |
